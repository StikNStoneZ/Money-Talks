# Money Talks

> More Money? No Problem.

I wanted to make something that answers a question most people have at some point:

**"What actually happens if I keep spending like this?"**

Money Talks is a simple financial simulator that compares three different lifestyles -- saving a little more, spending normally, or spending a little too much -- and shows how each one affects your savings over time.

It's not meant to replace financial advice. It's just a fun way to visualize how small decisions can snowball into very different futures.

---

## What it does

- Asks a few questions about your income, expenses and investments.
- Simulates three financial scenarios.
- Accounts for investment returns and inflation.
- Estimates how long it would take to reach your financial goal.
- Gives you a side-by-side comparison so you can see where your money is actually going.

## Running it

Clone the repository

```bash
git clone https://github.com/StikNStoneZ/Money-Talks.git
```

Run the Spring Boot application

```bash
mvn spring-boot:run
```

Open the frontend and start answering questions.

## Why I made this

This project started because I wanted to actually utilise the knowledge I earned from two semesters worth of Economics (both Macro and Micro) that I had to do for my Computer Science and Information Technology course.

I was really interested to implement in code financial strategies, simulations, inflation calculations, investment growth, and a UI that felt more like an app than a form.

It also ended up being a good excuse to learn how to connect a Java Spring Boot backend with a JavaScript frontend.
